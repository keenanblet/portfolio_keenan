#  Portfolio - Keenan Blet

## Présentation du projet

Ce projet a été réalisé dans le cadre du module **Projet Transversal B1 – EPSI**.
L’objectif est de concevoir un portfolio web personnel afin de présenter mes compétences, mes projets et mon profil.

---

## Parcours choisi

**Parcours B – Développement Web**

J’ai choisi ce parcours car je suis particulièrement intéressé par le développement web, le design et l’expérience utilisateur.

---

## Technologies utilisées

* HTML5
* CSS3
* Tailwind CSS
* Font Awesome
* GSAP

---

## Hébergement

Le site est hébergé via **GitHub Pages**.

### Fonctionnement :

* Les fichiers du site sont stockés sur GitHub
* GitHub Pages agit comme un serveur web
* L’utilisateur accède au site via une URL publique

---

## Contenu du site

Le site contient les pages suivantes :

* Accueil
* Compétences
* Projets
* Contact
* Infrastructure (explication du fonctionnement)

---

## Schéma d’infrastructure

Utilisateur → Navigateur → Internet → GitHub Pages → Fichiers du site

---

## Maquette

Une maquette simple a été réalisée en amont afin de définir :

* la structure du site
* l’organisation des pages
* l’expérience utilisateur

---

## Difficultés rencontrées

* Mise en page responsive avec Tailwind
* Alignement des éléments (cards, icônes)
* Compréhension du déploiement avec GitHub Pages

---

## Améliorations possibles

* Ajouter du JavaScript pour plus d’interactivité
* Améliorer les animations et transitions
* Ajouter plus de projets
